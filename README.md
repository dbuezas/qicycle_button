# QiCycle Button

For the DigisSpark - ATTiny85

## Warning:
* This version is untested!
* Check your local regulations before using this.
* Use at own risk

## DigiSpark - ATTiny85
![](digispark.png)

## Button

* GND   -> cable 1
* PIN 0 -> cable 2 (it doesn't matter which is which)

## PAS:

* VIN   -> (+) 
* GND   -> (-) 
* PIN 2 -> (data) 

![Thanks @mBurgas for the diagram!](pinout_diagram.png)
